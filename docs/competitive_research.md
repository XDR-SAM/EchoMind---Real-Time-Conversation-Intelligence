# Competitive Research — Meeting Intelligence Apps

This report benchmarks EchoMind against established meeting intelligence products to clarify differentiation opportunities and priority feature gaps.

## Competitive Landscape

| Product | Differentiator | Local/Offline | Platform | Pricing Model |
|---|---|---|---|---|
| Otter.ai | Live transcription, meeting notes, action items | No | Web, mobile | Subscription |
| Fireflies.ai | Recording, transcription, search, CRM integration | No | Web, mobile | Subscription |
| Tactiq | Chrome extension, live captions, highlights | No | Chrome extension | Freemium |
| Grain | Clip sharing, highlight reels, search | No | Web, mobile | Subscription |
| Krisp | Noise suppression, meeting transcription | Partial | Desktop, mobile | Freemium |
| EchoMind | Local inference, private, system-audio overlay, RAG over docs | Yes | Windows desktop | MIT / commercial |

## User Comparisons They Actually Make

1. Transcription accuracy compared to cloud STT
2. Latency and responsiveness
3. Privacy and offline capability
4. Action-item extraction quality
5. Export flexibility
6. Integration with existing tools
7. Pricing transparency

## EchoMind Differentiation Opportunities

1. **Privacy-first** — fully local inference, no cloud transcription, no telemetry
2. **RAG over docs** — retrieve context from local documents
3. **Windows system audio** — loopback capture without browser extension
4. **Open source / MIT** — no SaaS lock-in, no recurring cost for hosted transcription
5. **Low-latency offline** — no network dependency during meetings

## Priority Feature Recommendations

### Must Have
- Session persistence and transcript history
- Structured action-item extraction
- Multiple export formats

### Should Have
- Multi-speaker diarization
- Analytics and usage insights
- Cross-platform support

### Nice to Have
- Calendar integration
- Plugin/API surface
- Advanced privacy controls

## Market Positioning

EchoMind should position as the **privacy-first, offline-capable Windows meeting copilot** for users who need local AI assistance without cloud dependency. The primary market is:
- Developers and power users
- Privacy-conscious professionals
- Enterprise environments with data restrictions
- Users with unreliable internet during meetings
